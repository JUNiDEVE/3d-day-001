# 🌊 Day 001 — Deep Sea Bioluminescence

**Date:** May 26, 2026  
**Theme:** Deep Ocean · Bioluminescent Creatures  
**Tech:** WebGL (GLSL Fragment Shader) — zero dependencies

## What it is
A real-time raymarched deep-sea environment rendered entirely in a WebGL fragment shader. You are 3,800 meters below the ocean surface, surrounded by bioluminescent jellyfish, plankton, vampire squids, anglerfish and more — all glowing with authentic oceanic colors.

## Features
- 🪼 3 animated jellyfish with glowing domes, rims & tentacles
- 🦑 6 large deep-sea creatures with trailing tendrils
- ✨ 80 bioluminescent plankton particles
- 🌊 Fractal Brownian Motion water caustics
- 🖱️ Mouse parallax — move to swim through the water
- 🖱️ Scroll to dive deeper (depth meter updates live)
- 📟 Live creature detector panel
- 🎬 All in a single HTML file — no libraries
